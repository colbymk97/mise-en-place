# planning/

Agents for turning ideas into structured plans and specs.

## Agents

### [idea-elaboration](idea-elaboration.md)

Walks a rough project idea through 11 structured sections to produce AI-implementable artifacts — project brief, entity map, interface contracts, and a phased task list.

**The process:** Core Idea Capture, Use Cases, Edge Cases, Actors & Permissions, Data Model, Integrations, Non-Functional Requirements, Open Questions, Architecture, Implementation Roadmap, Output Artifacts

**Key trait:** One section at a time. The agent asks clarifying questions at each step and calibrates depth to the project's complexity — a weekend hack gets a light pass, an enterprise platform gets the full treatment.
